# 🤖 GitHubBot

Un bot básico hecho en **Python** que envía mensajes de prueba a Telegram.

## 🚀 Requisitos
- Python 3.8+
- pip
- Flash

## ⚙️ Instalación
Clona el repositorio e instala dependencias:

```bash
git clone https://github.com/verushkab/GitHubBot.git
cd GitHubBot
pip install -r requirements.txt
